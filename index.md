# GEO Launch Kit

Technical validation and reproducibility repository for the GEO framework.

This repository contains:

- CLASS v3.x perturbation patches,
- reproducible fitting scripts,
- diagnostic utilities,
- observational comparison tools,
- and numerical sanity checks.

## Purpose

The GEO Launch Kit is intended exclusively for:

- technical verification,
- reproducibility analysis,
- perturbation inspection,
- and computational cosmology review.

This repository does not represent an independent theory publication.

The associated theoretical framework is available here:

👉 https://github.com/LeoTorreblanca/GEO-hidden-geometry-framework

Public OSF registry:

👉 https://osf.io/bq4zg/
